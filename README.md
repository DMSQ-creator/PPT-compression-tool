编写一个既能保持高清晰度又能大幅压缩体积的PPT工具，核心原理是对PPT内部的**图片资源**进行智能处理。

PPT文件本质上是一个压缩包（ZIP），体积大通常是因为里面包含了分辨率过高（例如4000px宽度的照片放映时只需要1920px）的图片。

我为你提供一个基于 **Python** 的解决方案。它包含一个图形界面，操作简单，并且使用了智能算法：**仅调整分辨率过大的图片，同时保持视觉上的高清晰度**。

### 准备工作

你需要安装 Python 和必要的图片处理库。
在终端或命令行中运行：
```bash
pip install pillow
```


### 这个工具为什么能满足你的要求？

1.  **智能缩放 (Smart Resizing)**:
    *   **原理**: 很多 PPT 里的图片是直接从单反相机或素材网拖进去的，宽度可能达到 4000-6000 像素。但投影仪或电脑屏幕通常只有 1920 (1080P) 或 2560 (2K) 像素。
    *   **本工具做法**: 代码中设定了 `max_width=2048`。如果图片宽于这个数值，它会用 `LANCZOS` 算法（这是图像缩放中清晰度最高的算法之一）将其缩小到 2048 像素。**这对肉眼来说清晰度不变，但体积能减少 50%-80%。**

2.  **视觉无损压缩 (Visually Lossless)**:
    *   **原理**: JPG 图片在质量 100 和 质量 85 之间，肉眼几乎无法分辨噪点，但文件大小相差巨大。
    *   **本工具做法**: 代码设定 `quality=85` 并开启 `optimize=True` 选项，去除图片元数据并优化哈夫曼编码，在不产生可见噪点的前提下压榨体积。

3.  **透明度保护**:
    *   针对 PNG 图片，脚本会检测是否包含透明通道。如果有，它不会强制转为 JPG（否则背景会变黑），而是保持 PNG 格式但进行算法优化。
